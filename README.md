# Chord Gen

AI-powered chord progression generator. Describe a mood, genre, or vibe and get musically valid progressions you can play, edit, and export.

**[chordgen.org](https://chordgen.org)**

## Stack

- Next.js 16, React 19, TypeScript
- Vercel AI SDK
- tonal.js · Tone.js · midi-writer-js

## Local Development

```bash
# Install dependencies
pnpm install

# Set up environment
echo "DEEPSEEK_API_KEY=your_key" > .env.local

# Start dev server
pnpm dev
```
